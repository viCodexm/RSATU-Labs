действие пройграммы \ действие пользователя

1. выводит сообщение:
введите границы диапазона a < b
a =         // ввод a
b =         // ввод b

2. если a < b - переход к шагу 3
иначе - вывод сообщения:
"ошибка ввода границ диапазона" - переход к шагу 1

3.
"введите число C, принадлежащее заданному диапазону"
c =         // ввод c

4. если C принадлежит диапазону - переход к шагу 5
иначе - вывод сообщения "ошибка ввода числа C" - возврат к шагу 3

5. программа выводит сообщение:
"введите номер вида кода:
1 Прямой код
2 Обратный код
3 Дополнительный код
4 Модифицированный дополнительный код
номер вида кода = "     // ввод номера вида кода

6. если введено не число 1-4 - вывод сообщения
"ошибка ввода номера вида кода" и возврат к шагу 5
если введено верно - переход к шагу 7

7. вывод сообщения
"название вида кода"
"C = (value)" значение в соответсвующем коде

8. вывод сообщения
"изменить номер вида кода?"
"Y-да, N-нет"              // ввод символа

9. если введено не Y и не N 
"ошибка запроса изменения вида кода" - переход на 8 шаг
если введено Y, то возврат к 5 шагу
если введено N, то переход к 10 шагу

10. вывод запроса
"ввести новое число C?"
"Y-да,N-нет"        // ввод пользователя

11. если введено не Y и не N 
"ошибка запроса на ввод нового числа" возврат к этому запросу
если введено Y - возврат к шагу 3
если нет N - переход на 12 шаг

12. "завершить работу программы?"
"Y-да,N-нет"        // ввод пользователя

13. если не Y и N, то "ошибка ввода запроса на завершение программы" возврат к этому запросу
Y - закрывается
N - возврат к шагу 1
